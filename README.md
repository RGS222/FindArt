This is the second capstone project as part of the "Complete Full-Stack Web Development Bootcamp" by
Dr. Angela Yu at Udemy.

The goal of this project is to create a website using Node.js, Express.js, axios and EJS accessing
a public API.  The API from the Metropolitan Museum of Art was chosen.  This project has demonstrated 
the use of EJS for server-side rendering and the interaction with Javascript altering the web page 
element appearances as well as the passing of data to and fro the server and the API.  The web page 
appearance and the user interaction are also import aspects in this project.

To run this website, you will need to have vscode, node, and npm installed.  Then,
- Use "npm install", at the root folder, to get the dependencies installed.
- Run the website with "node index.js".
- Open Chrome and goto address "http://localhost:3000/", to access the website.
